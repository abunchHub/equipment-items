Предметы экипировки и ценности

После завершения сюжетного задания Путь бессмертного станут доступны предметы экипировки и ценности в приключениях и на местностях.

Предметы экипировки

За уничтожение противника-босса выпадает предмет экипировки необычного качества. За освобождение провинции (выполнение всех заданий в ней) — предмет редкого качества.

Однако в случае с драгоценностями и камнями силы важно не столько качество предмета, сколько его главные свойства и их диапазон значений.

Что такое качество в предмете экипировки?

Качество отвечает за количество свойств в предмете экипировки:

  • [необычное] — 2 главных свойства
  • [редкое] — 2 главных и 1 дополнительное свойство.

Что такое свойство в предмете экипировки?

Свойство — эффект, который добавляет или усиливает характеристики персонажа. Диапазон значений указан в скобках, выбирать стоит с максимальным показателем. Отметим свойства для драгоценностей и камней силы в порядке приоритета.

Драгоценности:

  • [первый приоритет] урон по элите (4,0–6,0%)
  • [второй приоритет] бонус к основному урону (5,0–7,0%).
  • [третий приоритет] урон по миньонам (4,0–6,0%)
  • [четвертый приоритет] шанс критического урона (4,8–7,2%)

Камни силы:

  • [первый приоритет] бонус к основному урону (8,0–12,0%).
  • [второй приоритет] бонус к критическому урону (8,0–12,0%)
  • [третий приоритет] шанс критического урона (2,4–3,6%)
Бессмертный
Не совсем понимаю, по какому принципу выбирать свойства в предмете экипировки?

Всё просто — выбрать следует тот предмет экипировки, у которого на первом месте стоит свойство высшего приоритета. В драгоценности — урон по элите, чтобы эффективнее уничтожать противника-босса, а в камне силы — бонус к основному урону, так как это даст стабильный урон.

Это не все свойства. Сейчас указаны только те, которые пригодятся на текущий момент.

Бессмертный
Остальные предметы экипировки продавать?

Да, но не все. Если полный набор из четырех драгоценностей, в каждой из которых есть свойство «урон по элите», уже собран, а в сумке лежит набор драгоценностей со свойством «урон по миньонам», это значит, что есть по сути еще один готовый комплект экипировки, подходящий для определенных задач.

Бессмертный
Что такое комплект экипировки?

Для разных задач могут потребоваться предметы с разными свойствами. Например, для битвы с боссом стоит использовать комплект со свойствами «урон по элите» и «бонус к основному урону», описанный выше. А для сражения с обычными противниками и зачистки местности — второй, со свойством «Урон по миньонам» у драгоценностей.

Камни силы можно использовать из первого комлпекта.

На экране «Экипировка», между слотов для оружия и артефакта можно увидеть панель с шестью цифрами — это и есть различные комплекты снаряжения. Можно выбрать необходимые предметы для каждого варианта и быстро менять их вне боя.
